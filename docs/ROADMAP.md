# Roadmap do Momentum

Este roadmap mantém a evolução do projeto organizada e educativa. Cada fase deve introduzir novos conceitos gradualmente, sem esconder decisões importantes de engenharia atrás de complexidade desnecessária.

## Fase 1: Fundação do Repositório

- Inicializar o repositório Git.
- Criar um README profissional.
- Adicionar um `.gitignore` útil.
- Definir a visão inicial do projeto.
- Criar o primeiro repositório no GitHub.

## Fase 2: Fundação do Frontend

- Criar a aplicação React.
- Definir a estrutura base de pastas.
- Construir o primeiro layout de dashboard.
- Estabelecer estilos globais e direção visual em dark mode.
- Criar componentes reutilizáveis de interface.

## Fase 3: Fundação do Backend

- Criar a API com Node.js e Express.
- Organizar rotas, controllers e services.
- Adicionar endpoint de health check.
- Conectar SQLite.
- Criar o primeiro recurso de acompanhamento.

## Fase 4: Primeiras Funcionalidades de Produto

- Controle de hábitos.
- Controle de hidratação.
- Controle de sono.
- Controle de estudos.
- Métricas básicas no dashboard.

## Fase 5: Profissionalização

- Melhorar README e documentação.
- Adicionar testes.
- Adicionar autenticação.
- Melhorar tratamento de erros.
- Adicionar gráficos e analytics.
- Preparar estratégia de deploy.
