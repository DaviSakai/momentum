# Frontend do Momentum

Esta pasta contém a aplicação web do Momentum.

## Stack

- React
- JavaScript
- Vite
- CSS moderno

## Estrutura

```text
frontend/
├── public/
├── src/
│   ├── App.jsx
│   ├── main.jsx
│   └── styles.css
├── index.html
├── package.json
└── vite.config.js
```

## Scripts

Depois de instalar as dependências, use:

```bash
npm run dev
```

## Objetivo Inicial

A primeira versão do frontend apresenta um dashboard estático em dark mode para definir a direção visual do produto antes da integração com backend e banco de dados.
